Documented weak spots

These are pulled directly from grading marks on Prelim 1 and Prelim 2. High probability of reappearing.

Trap 1 — actual vs. maximum static friction Prelim 2 Q1c

f_s = mg sinθ when block is stationary (from Newton's 2nd law). f_s,max = μ_s·mg·cosθ is the maximum before sliding. These are DIFFERENT unless the block is on the verge of slipping.

Rule: if asked for friction on a stationary object, use ΣF = 0 to find it. Only use μ_s·N when the question asks "when does it slip?"

Trap 2 — energy when non-conservative forces act Prelim 2 Q5

Block gently lowered by hand → hand does NEGATIVE work → E_final < E_initial. Block dropped from same height → E_final = E_initial. The hand removes energy from the system.

Always identify W_other in ΔK + ΔU = W_other. If a hand, motor, or friction is involved, mechanical energy is NOT conserved.

Trap 3 — belt/gear wheel ratios Prelim 2 Q11

Belt connects wheels: rim speeds are equal → ω_L·r_L = ω_s·r_s → ω_L = ω_s·(r_s/r_L). LARGER wheel spins SLOWER. If r_L = 2r_s then ω_L = ω_s/2.

Trap 4 — momentum conservation: when? Prelim 2 Q13

The F(t) graph between two blocks shows an INTERNAL force. For the two-block SYSTEM, momentum IS conserved because there are no external horizontal forces. You answered "No" — wrong.

Internal forces don't change system momentum. Always ask: what is my system? What external forces act on it?

Trap 5 — drag force & terminal velocity Prelim 2 Q2

At terminal speed: drag = weight, net force = 0, a = 0. Moving faster than terminal: drag > weight, net force upward, a < g. At point A (v = 0): drag = 0, a = g downward.

Trap 6 — 2D vector direction after correct magnitude Prelim 2 Q14f

You correctly calculated the speed but drew the wrong direction. Always extract BOTH p_x and p_y components, determine the quadrant, THEN draw the arrow.

Trap 7 — Newton's 2nd parallel component drops gravity Prelim 1 Q13e

For Ana on slope: F_BonA − M_A·g·sinθ = M_A·a → F_BonA = M_A(g sinθ + a). You wrote only M_A·a. Never drop forces from Newton's 2nd.

Trap 8 — monatomic vs diatomic C_V Thermo

Monatomic (He, Ar): C_V = 3R/2, C_P = 5R/2, γ = 5/3. Diatomic (N₂, O₂): C_V = 5R/2, C_P = 7R/2, γ = 1.40. Most thermo cycle problems use diatomic.

Key conservation problems

Acceleration

Derived via Στ = dL/dt. M/2 appears because I_pulley = (1/2)MR² and the constraint gives (1/2)M effective mass. If M → 0, recovers massless pulley result.

Tipping stick (COOP 7B)

Angular speed at angle θ
ω(θ) = √[3g(1−cosθ)/D]
Angular acceleration at θ
α(θ) = 3g sinθ / (2D)
Tip speed at ground

How to get α(θ): use chain rule α = (1/2)d(ω²)/dθ. From ω² = 3g(1−cosθ)/D, differentiate: d(ω²)/dθ = 3g sinθ/D, so α = 3g sinθ/(2D).

End moves faster than free-fall because pivot constrains lower end — upper end must travel a longer arc. Energy conserved because pivot does no work (pivot doesn't move).

Precession (COOP 9C)

Precession rate

  • Faster spin → larger L → smaller angle swept per dt → SLOWER precession

Four thermodynamic processes

Process table — know this cold

Auto engine: V₂ = 0.09V₁, γ = 1.40 → T₂ = 293·(11.1)^0.40 = 768 K. Verify with pV = nRT at each state.

Multi-step cycle strategy

1
Draw pV diagram, label all states (1, 2, 3 …)
2
Identify each process: isochoric (V const), isobaric (p const), isothermal (T const), adiabatic (Q=0)
3
Use pV = nRT to find T, p, V at each state you don't know
4
Apply the process table row for W, Q, ΔU at each step
5
Verify ΔU = Q − W at each step as a self-check
6
W_net = sum of all W. Q_H = sum of positive Q only. Q_C = sum of negative Q only.
7
e = W_net/Q_H. Compare to e_Carnot = 1 − T_C/T_H (Kelvin!)

Free expansion & irreversibility (HW 12)

  • Gas expands freely into vacuum: W = 0, Q = 0 → ΔU = 0 → T unchanged
  • Then isothermal compression back: W_on_gas = nRT·ln(2), heat flows OUT = nRT·ln(2)
  • Net: surroundings did work and received heat — NOT returned to original state
  • This is irreversible — entropy increased during free expansion
  • 2nd Law: cannot return BOTH gas and surroundings to original states simultaneously
If gas had expanded isothermally against a piston instead: W_by_gas = nRT·ln(2) extracted usefully, process fully reversible — everything can be undone.

Heat engines & second law

Heat engine fundamentals

First law for engine
Q_H = W + |Q_C|
Efficiency
e = W/Q_H = 1 − |Q_C|/Q_H
Carnot (maximum) efficiency
e_Carnot = 1 − T_C/T_H
Refrigerator COP
K = |Q_C|/W ≤ T_C/(T_H−T_C)
ALWAYS convert to Kelvin: T_C = 27°C = 300 K. No engine can exceed Carnot efficiency.

Checking inventor's engine

1
1st Law check: Does Q_H = W + |Q_C|? Convert all units (1 cal = 4.186 J, 1 kWh = 3.6×10⁶ J) then check energy balance.
2
2nd Law check: Is e = W/Q_H ≤ e_Carnot = 1 − T_C/T_H? If e > e_Carnot → impossible.
HW 12 example: Q_H = 3.0×10⁶ J, W = 1.26×10⁶ J, Q_C = 1.34×10⁶ J. Energy check: 1.26 + 1.34 = 2.60 ≠ 3.0 → violates 1st Law. Also e = 42% > e_Carnot = 33% → violates 2nd Law. Do not invest.

All 9 co-op answers

COOP 7B — Tipping stick

  • U_g(θ) = Mg(D/2)cosθ, zero when lying flat (θ = 90°)
  • KE = (1/6)MD²ω² — same by pure rotation and CM+rotation methods
  • ω(θ) = √[3g(1−cosθ)/D]
  • α(θ) = 3g sinθ/(2D) — derived via chain rule α = (1/2)d(ω²)/dθ
  • v_end at ground = √(3gD) > √(2gD) (free-fall speed)
  • a_end at ground = 3g/2 > g — pivot constrains lower end

COOP 8A — Whirlygig

  • Only string tension T produces net torque (weights cancel by symmetry — symmetric geometry required)
  • Net torque: Στ = TR (clockwise)
  • I = (1/2)MR² + (1/3)m_r·r² + 2m_s·r²
  • Constraint: a = αR (string inextensible on cylinder of radius R)
  • Block acceleration: a = mg/(m + I/R²)
  • T < mg because block accelerates downward
  • m → 0: a → 0 (heavy whirlygig barely moves). m → ∞: a → g (free fall).

COOP 8C — Monkey ladder

  • N_G = w + w_L (constant — doesn't change as monkey climbs)
  • N_W = cosθ·(w_L·L/2 + w·s) / (L sinθ) — increases as monkey climbs
  • Friction f_s = N_W (from horizontal equilibrium)
  • Max fraction monkey can climb: s/L = μ_s(w+w_L)tanθ/w − w_L/(2w)
  • Numerical (20 lb monkey, 16 lb ladder, 50°, μ_s = 0.50): s/L ≈ 0.67

COOP 9A — Sweet spot

  • Conserved: linear momentum, angular momentum about stick CM. KE is unknown (may be inelastic).
  • v_CM = m(v₀−v_f)/M — independent of impact parameter b
  • ω = 12mb(v₀+v_f)/(MD²)
  • Sweet spot condition: lower end velocity = 0 → v_CM = ω(D/2)
  • Result: b = D/6 from CM = 2D/3 from top end
  • Independent of all speeds and masses — depends only on geometry D

COOP 9C — Bike wheel precession

  • Gravity torque about P: τ⃗ = r⃗(→) × F⃗(↓) = ⊙ (out of page)
  • dL⃗ = τ⃗·dt → tip of L_spin sweeps CCW as seen from above
  • Precession rate: Ω = MgD/(Iω)
  • Faster spin → slower precession. Reversed spin → reversed precession.
  • Stop precession: push UP at B (right end), push DOWN at A (left end)

COOP 10A — Ball on hand

  • v_y(t) = 2πfA cos(2πft), v_max = 2πfA
  • a_y(t) = −(2πf)²A sin(2πft), a_max = (2πf)²A
  • N(t) = m[g − (2πf)²A sin(2πft)]
  • Contact lost at TOP of motion (y = +A, sin = +1) — N is minimum there
  • Critical: N = 0 → f_crit = (1/2π)√(g/A). For A = 0.25 m: f ≈ 1.0 Hz

COOP 10B — Board on rollers

  • Normal forces from torque balance: N_L = Mg(½+x/D), N_R = Mg(½−x/D)
  • Net force: F_x = −(2μ_kMg/D)·x → linear restoring force
  • k_eff = 2μ_kMg/D, ω₀ = √(2μ_kg/D)
  • Frequency independent of mass M (M cancels)
  • Decrease frequency: increase D (wider roller spacing) or decrease μ_k

COOP 11A — Styrofoam cooler

  • Ice melts at constant 0°C (latent heat). Total capacity before T rises: Q = mL_f.
  • One wall heat flow: H = kD²ΔT/w. Six faces total: H_total = 6kD²ΔT/w
  • Required thickness for time t: w = 6kD²ΔT·t/(mL_f)
  • Cases 1 & 2 (ice at wall): highest heat flow, uniform 0°C throughout
  • Case 3 (ice center, soda surrounding): lowest heat flow to ice → ice lasts longest
  • Larger cooler lasts longer: volume ∝ D³ but area ∝ D², so t ∝ D

COOP 11B — Solar radiation & greenhouse

  • P_sun = S₀·4πd² = 3.87×10²⁶ W. T_sun = [P/(4πR_S²σ)]^(1/4) ≈ 5800 K
  • Earth equilibrium: T_E = [S₀(1−α)/4σ]^(1/4) = 255 K (predicted) vs 288 K (actual)
  • Difference = 33°C greenhouse effect
  • Snowball Earth (α=0.70): T_E = 218 K — two stable equilibria with same solar input

Thermodynamic cycle — full worked example

HW 11 oxygen cycle. Diatomic gas (n = 0.250 mol), C_V = 5R/2, C_P = 7R/2.

State 1: p₁ = 2.40×10⁵ Pa, T₁ = 355 K

Process 1→2: isobaric expansion to V₂ = 2V₁

  • T₂ = T₁·(V₂/V₁) = 2 × 355 = 710 K
  • W₁₂ = nRΔT = (0.250)(8.31)(355) = 738 J
  • Q₁₂ = nC_P·ΔT = (0.250)(7/2)(8.31)(355) = 2580 J
  • ΔU₁₂ = Q − W = 2580 − 738 = 1840 J ✓ (= nC_V·ΔT)

Process 2→3: isothermal compression back to V₁ at T = 710 K

  • ΔU₂₃ = 0 (isothermal, ideal gas)
  • W₂₃ = nRT·ln(V₁/2V₁) = −nRT·ln(2) = −(0.250)(8.31)(710)(0.693) = −1020 J
  • Q₂₃ = W₂₃ = −1020 J (heat flows out)

Process 3→1: isochoric cooling back to T₁ = 355 K

  • W₃₁ = 0 (constant volume)
  • Q₃₁ = nC_V·ΔT = (0.250)(5/2)(8.31)(355−710) = −1840 J
  • ΔU₃₁ = Q − W = −1840 − 0 = −1840 J

SHM derivation template

Use this for ANY novel oscillator on the exam. The physical setup will change — the method doesn't.

Universal method

1
Draw extended FBD at equilibrium (x = 0). Verify ΣF = 0 — this confirms equilibrium.
2
Displace by small x (or angle θ). Redraw FBD showing how forces change with displacement.
3
Express all forces in terms of x. Use torque balance if normal forces change (like board on rollers).
4
Apply Newton's 2nd: Ma = ΣF(x). Show this takes the form Ma = −C·x where C > 0.
5
Identify: ω₀² = C/M. Write ω₀, f = ω₀/(2π), T = 2π/ω₀.
6
Check: does mass cancel? Is ω₀ in rad/s? Do limiting cases make sense (C → 0 gives T → ∞ etc.)?

For physical pendulum (and any rotational SHO)

1
Net torque at small angle: τ_net = −Mgd·sinθ ≈ −Mgd·θ (small angle approximation)
2
Newton's 2nd for rotation: Iα = −Mgd·θ
3
Rearrange: d²θ/dt² = −(Mgd/I)·θ → ω₀² = Mgd/I
4
ω₀ = √(Mgd/I), T = 2π√(I/Mgd). Equivalent simple pendulum length: L_eq = I/(Md).

Free response checklist

For any mechanics problem

1
Draw FBD — extended (at correct location) if rotation is involved
2
Choose coordinate system — label positive direction explicitly
3
List ALL forces — don't drop gravity, normal forces, or tension
4
Write ΣF = ma (translation) AND Στ = Iα (rotation) separately
5
Write constraint equations (rolling: a = αR, string: same speed both ends)
6
Identify conserved quantities and state WHY they are conserved
7
Solve symbolically — plug in numbers at the very end
8
Check: units, signs, limiting cases, physical reasonableness

For thermodynamics cycle problems

1
Identify gas type → get C_V, C_P, γ
2
Draw pV diagram, label all states
3
Identify each process (const V, p, T, or adiabatic)
4
Use pV = nRT to find all unknown state variables
5
Apply process table: W, Q, ΔU for each step
6
Verify ΔU = Q − W at each step
7
Sum for cycle: W_net, Q_H (positive Q only), e = W_net/Q_H
8
Compare to e_Carnot = 1 − T_C/T_H (Kelvin!)
